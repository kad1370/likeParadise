# Like Paradise

## 소개
Like Paradise 는 호텔 소개 및 예약 사이트 입니다.
SpringBoot 프레임워크 기반으로 개발되었습니다.

## 개발환경
 * OS: window10
 * IDE: eclipse
 * FrameWork : Spring 4.2.1.RELEASE
 * DB: mariaDB
